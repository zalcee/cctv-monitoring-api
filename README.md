A robust and scalable RESTful API designed for managing, streaming, and monitoring CCTV camera feeds in real-time. This API allows developers to integrate surveillance management, health checks, and event-driven alerts into their applications.

🚀 Features
Camera Management: Easily register, update, and categorize CCTV cameras by location or department.

Live Stream Integration: Support for RTSP, HLS, or WebRTC streaming protocols.

Health Monitoring: Automated uptime, ping, and latency tracking for connected hardware.

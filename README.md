# CCTV Monitoring and Retention Management API

A centralized and scalable RESTful API designed to monitor CCTV systems across multiple store locations. The platform provides real-time visibility into NVR health, recording status, camera availability, and storage retention to help ensure continuous surveillance operations.

## 🚀 Features

### Camera & NVR Monitoring

* Manage and monitor CCTV cameras connected to NVRs across multiple stores.
* Organize cameras by store, location, or department.
* View camera status and connectivity information in real time.

### CCTV Collage Snapshot Generation

* Automatically capture screenshots from multiple NVR channels.
* Generate collage views that provide a quick visual overview of all camera feeds in a single image.
* Enable remote verification of camera functionality without accessing the NVR directly.

### Recording Retention Monitoring

* Retrieve and display available recording retention days for each NVR.
* Monitor storage usage and recording history across all locations.
* Detect insufficient retention periods caused by storage limitations or system issues.

### Recording Gap Detection

* Identify dates and time periods where CCTV recording is missing.
* Generate alerts when cameras or NVRs fail to record as expected.
* Help ensure compliance with surveillance and security policies.

### System Health Monitoring

* Monitor NVR availability, uptime, response time, and network connectivity.
* Detect offline devices and communication failures.
* Provide status dashboards for proactive maintenance and troubleshooting.

## 🎯 Purpose

The primary purpose of this system is to provide centralized monitoring of CCTV infrastructure across all store locations. It enables IT and security teams to quickly verify camera availability, review collage snapshots of camera channels, monitor recording retention periods, and identify recording gaps to ensure continuous and reliable surveillance coverage.
